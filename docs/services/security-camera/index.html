Security

防犯関連

ヨリハブは、防犯カメラや住まいの防犯対策について相談受付・内容整理・案内を行う窓口です。 現地確認、見積、設置工事などの実務は、提携先事業者が担当します。

役割分担について

ヨリハブが担当すること

  • ✓ご相談の受付・一次対応
  • ✓ご要望・状況の内容整理
  • ✓対応可能な案内の提供
  • ✓提携先事業者へのつなぎ・紹介

提携先事業者が担当すること

  • ▸現地確認・現場調査
  • ▸機器の選定・提案
  • ▸見積の作成・提示
  • ▸実際の設置工事・施工

ご相談できる内容

以下のようなご相談をお受けしています。まずは状況をお聞かせください。

防犯カメラの設置を検討している

自宅・事業所・駐車場など、どこにどのようなカメラが必要かを整理するお手伝いをします。

どんな対策が必要かわからない

現状のご不安や状況をお聞きし、一般的な対策の概要をお伝えします。

業者の選び方・進め方を知りたい

施工会社への依頼の流れや、確認すべき点について案内します。

見積を取りたいが何社に聞けばよいかわからない

提携先の事業者をご紹介できる場合があります。詳細はお問い合わせください。

ご相談から案内までの流れ（概要）

※ 以下はヨリハブが関与する範囲の一般的な流れです。提携先事業者の対応フローは各事業者によります。

  1. 1

    お問い合わせ

    お問い合わせページよりご連絡ください。状況・ご要望をお知らせください。

  2. 2

    内容確認・折り返し

    いただいた内容を確認し、必要に応じて追加のご確認をさせていただきます。

  3. 3

    案内・紹介

    状況に応じて、対応可能な内容をお伝えします。提携先事業者のご紹介が適切な場合はその旨をお伝えします。

  4. 4

    提携先とのやり取り

    提携先事業者への引き継ぎ後は、現地確認・見積・施工の対応は提携先が担当します。

案内時に大切にしていること

誇大な表現を使いません

「絶対安全」「完全対応」などの表現は使いません。できること・できないことを正直にお伝えします。

費用の見通しは提携先から

ヨリハブが具体的な費用を提示することはありません。費用・見積は提携先事業者から正式にご提示します。

急かしません

ご検討のペースに合わせて対応します。無理な契約を促すことはありません。

役割の境界を明確に

ヨリハブの対応範囲と提携先の対応範囲を事前にお伝えします。曖昧な状態で手続きを進めません。

防犯カメラの相談をご希望の方へ

防犯カメラの設置や見直しをご検討中の方は、専用の案内ページをご確認ください。 ご相談の流れや対象内容を確認したうえで、必要に応じてご利用いただけます。

防犯カメラの案内ページを見る

事業内容・提携に関するお問い合わせは、お問い合わせページをご利用ください。
